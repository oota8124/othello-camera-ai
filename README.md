# オセロカメラAI

オセロ盤面を撮影するだけで、AIが最善手を提案するiOSアプリです。

## 主な機能

- 📸 カメラで盤面を自動認識
- 🤖 AIが最善手を3候補提案
- 📊 形勢グラフで優劣を可視化
- 💡 AI解説で戦略を学習
